## わかったこと
- cat /ファイル名 → ファイルを開く
- less /ファイル名 → スクロールしながらファイルの表示
- ln リンク元ファイル名 -s ファイル名 → リンクを貼る
- cat < 入力ファイル > 出力先ファイル → catの結果を出力ファイルに入れる
- cat ファイル名 | less → |(パイプライン)はコマンドを連携させる
- ps → プロセスの表示(auxをつけると全詳細を表示)
- jobs → 現在のjob一覧の表示
- fg %ジョブID → ジョブをフォアグランドにする
- bg %ジョブID → ジョブをバックグランドにする
